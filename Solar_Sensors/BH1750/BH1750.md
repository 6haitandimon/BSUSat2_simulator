Driver to BH1750 with stm32.<br>

## How to use
```c
BH1750_t obj = BH1750_Init(&hi2c1,true); //generate object
uint8_t tmp;
 while (1)
  {
     tmp = BH1750_get_value(&obj); // get lx value
  }
```
